.. _actions_sdk:

actions sdk Documentation
#########################

Welcome to the Actions Project's :abbr:`actions sdk (Actions Software Development Kit)`
documentation.

Actions sdk is based on zephyer rots development, providing wifi, Bluetooth and other network services,
while supporting music playback, recording and other services.
Users can build their own products based on actions sdk,
such as network story box, Bluetooth speakers, etc.

The actions sdk function are used the same way on all actions platfrom.
